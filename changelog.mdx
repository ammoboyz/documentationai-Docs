## Upgrading Tyler Durden

Follow these steps to upgrade safely.

<Steps>
  <Step title="Backup" icon="database">

    Save your current configuration and docs.

  </Step>
  <Step title="Update Package" icon="package">

    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install tyler-durden@latest
    ````

    ````bash
    yarn add tyler-durden@latest
    ````
    </CodeGroup>

  </Step>
  <Step title="Review Changes" icon="book-open">

    Check this changelog for migration notes.

  </Step>
  <Step title="Test Locally" icon="play">

    Run `npm run dev` and verify your site.

  </Step>
</Steps>

<Callout kind="tip">
  Pin versions in `package.json` for production stability.
</Callout>
