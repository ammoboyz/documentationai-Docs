## Prerequisites

Before you begin, ensure you have the following:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: A GitHub account for importing repositories

</Callout>

## Create Your Account

Sign up for Tyler Durden in under a minute. Tyler Durden hosts your documentation spaces securely.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://dashboard.example.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Your Details" icon="user">
    Provide your email, create a strong password, and verify your account via email.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Accept the terms and select your workspace type (personal or team).
  </Step>
</Steps>

## Set Up Your First Project Space

Once logged in, create a dedicated space for your project documentation.

<Steps>
  <Step title="Create New Space" icon="plus">
    Click **New Space** in the dashboard and name it, e.g., `my-project-docs`.
  </Step>
  <Step title="Configure Basics" icon="settings">
    Set the brand color to `#3B82F6` and add a description: "Project documentation hub."
  </Step>
  <Step title="Publish Space" icon="globe">
    Save and publish. Your space is now live at `https://docs.example.com/my-project-docs`.
  </Step>
</Steps>

## Import Initial Documentation

Populate your space quickly with existing content.

<Tabs>
  <Tab title="Markdown Files" icon="file-text">
    Upload local `.mdx` files directly.
  </Tab>
  <Tab title="GitHub Repo" icon="github">
    Connect your repo for automatic sync.
    
    ```bash
    # Example GitHub import via CLI (optional)
    npx tyler-durden import --repo owner/my-docs --space my-project-docs
    ```
  </Tab>
  <Tab title="API Import" icon="zap">
    Use the API for bulk import.
    
    <CodeGroup tabs="cURL,JavaScript">
      ```bash
      curl -X POST https://api.example.com/v1/spaces/my-project-docs/import \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -d '{"files": [{"path": "README.mdx", "content": "# Welcome"}]}'
      ```
      ```javascript
      const response = await fetch('https://api.example.com/v1/spaces/my-project-docs/import', {
        method: 'POST',
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          files: [{ path: 'README.mdx', content: '# Welcome' }]
        })
      });
      ```
    </CodeGroup>
  </Tab>
</Tabs>
